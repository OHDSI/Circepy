### Cohort Entry Events

People may enter the cohort when observing any of the following:

1. condition occurrence of 'Acute Hepatic Failure' for the first time in the person's history.

### Inclusion Criteria

#### 1. Chronic Hepatic Failure

Entry events having no condition occurrences of 'Chronic Hepatic Failure', starting between 0 days before and 0 days after cohort entry start date.

#### 2. Liver Cirrhosis

Entry events having no condition occurrences of 'Liver Cirrhosis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 3. Viral Hepatitis

Entry events having no condition occurrences of 'Viral hepatitis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 4. Liver Fibrosis

Entry events having no condition occurrences of 'Liver Fibrosis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 5. Toxic Liver Disease (7d)

Entry events having no condition occurrences of 'Toxic Liver Disease', starting anytime up to 7 days before cohort entry start date; allow events outside observation period.

#### 6. Hepatic Necrosis (7d)

Entry events having no condition occurrences of 'Hepatic Necrosis', starting anytime up to 7 days before cohort entry start date; allow events outside observation period.

#### 7. Alcohol use disorder

Entry events with all of the following criteria:

   1. having no condition occurrences of 'Disorder caused by alcohol', starting between 365 days before and 0 days before cohort entry start date; allow events outside observation period.
   2. having no observations of 'Disorder caused by alcohol', starting between 365 days before and 0 days before cohort entry start date; allow events outside observation period.


#### 8. Alcoholic Hepatitis

Entry events having no condition occurrences of 'Alcoholic hepatitis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 9. Alcoholic Fatty Liver

Entry events having no condition occurrences of 'Steatosis of liver, not including non-alcoholic', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 10. Malignancies that spread to liver (breast, esophageal, stomach, pancreatic, lung, kidney, melanoma)

Entry events having no condition occurrences of 'Malignancies that spread to liver (breast, esophageal, stomach, pancreatic, lung, kidney, melanoma, primary malignancy of liver)', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 11. Chronic Hepatic Failure

Entry events having no condition occurrences of 'Chronic Hepatic Failure', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 12. Other Chronic Liver Disease

Entry events having no condition occurrences of 'Chronic Liver Disease', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 13. Ascites (7d)

Entry events having no condition occurrences of 'Ascites', starting anytime up to 7 days before cohort entry start date; allow events outside observation period.

#### 14. Portal Hypertension (7d)

Entry events having no condition occurrences of 'Portal Hypertension', starting anytime up to 7 days before cohort entry start date; allow events outside observation period.

#### 15. History of Liver Transplant

Entry events having no condition occurrences of 'Liver Transplant', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 16. Cholestasis (7d)

Entry events having no condition occurrences of 'Cholestasis', starting anytime up to 7 days before cohort entry start date; allow events outside observation period.

#### 17. Non-Alcoholic Fatty Liver Disease

Entry events having no condition occurrences of 'Non-Alcoholic Fatty Liver Disease', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 18. Autoimmune Hepatitis

Entry events having no condition occurrences of 'Autoimmune Hepatitis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 19. Primary Biliary Cholangitis

Entry events having no condition occurrences of 'Primary Biliary Cholangitis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 20. Hemochromatosis

Entry events having no condition occurrences of 'Hemochromatosis', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 21. Wilsons Disease

Entry events having no condition occurrences of 'Wilsons disease', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 22. Neoplasm of liver

Entry events having no condition occurrences of 'Neoplasm of liver', starting anytime on or before cohort entry start date; allow events outside observation period.

#### 23. Other liver problems entry event criteria: Note: this is a catch all  

Entry events having no condition occurrences of 'Other Liver problems', starting anytime up to 7 days before cohort entry start date; allow events outside observation period.

Limit qualifying entry events to the earliest event per person.

### Cohort Exit

The person exits the cohort at the end of continuous observation.

### Cohort Eras

Remaining events will be combined into cohort eras if they are within 0 days of each other.
