### Cohort Entry Events

People may enter the cohort when observing any of the following:

1. condition occurrences of 'Urinary bleeding not due to nephritic or nephrotic reasons'.

### Inclusion Criteria

#### 1. Exclude any event preceded by kidney biopsy in -7 to -1 days

Entry events having no procedure occurrences of 'Kidney biopsies', starting in the 7 days prior to cohort entry start date; allow events outside observation period.

#### 2. Exclude chronic kidney disease or dialysis

Entry events with all of the following criteria:

   1. having no condition occurrences of 'Chronic kidney disease', starting anytime on or before cohort entry start date; allow events outside observation period.
   2. having no procedure occurrences of 'Renal Dialysis', starting anytime on or before cohort entry start date; allow events outside observation period.


#### 3. Exclude any event that is preceded by renal or glomerular disease -365 to 0

Entry events with all of the following criteria:

   1. having no condition occurrences of 'Nephritic, nephrotic, glomerular disease', starting between 365 days before and 0 days before cohort entry start date; allow events outside observation period.
   2. having no observations of 'Nephritic, nephrotic, glomerular disease', starting between 365 days before and 0 days before cohort entry start date; allow events outside observation period.


### Cohort Exit

The cohort end date will be offset from index event's end date plus 365 days.

### Cohort Eras

Remaining events will be combined into cohort eras if they are within 0 days of each other.
