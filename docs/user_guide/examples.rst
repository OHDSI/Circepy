Examples and Use Cases
======================

Practical examples for common cohort definition scenarios.

Example Files
-------------

The ``examples/`` directory contains working Python scripts:

* **basic_cohort.py** - Simple single-condition cohort
* **complex_cohort.py** - Advanced multi-criteria cohort
* **generate_sql.py** - SQL generation techniques
* **validate_cohort.py** - Validation workflows

See the examples/ README for details on running these scripts.
