Testing
=======

CIRCE Python has comprehensive test coverage (71%, 896 tests).

Running Tests
-------------

.. code-block:: bash

   # Run all tests
   pytest

   # Run with coverage
   pytest --cov=circe

   # Run specific test file
   pytest tests/test_cohort_expression.py

Test Structure
--------------

Tests are organized by module in the ``tests/`` directory.

Writing Tests
-------------

Follow existing test patterns. See CONTRIBUTING.md for guidelines.
