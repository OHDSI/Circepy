Frequently Asked Questions
==========================

Is this compatible with OHDSI Atlas?
-------------------------------------

Yes, CIRCE Python can process cohort definition JSON files exported from Atlas.

Can I use this with CDM v5.3?
------------------------------

Yes, the package supports all OMOP CDM v5.x versions.

Does this replace Java CIRCE-BE?
---------------------------------

No, this is a complementary Python implementation. Both produce identical SQL output.

How do I convert camelCase JSON to Python?
-------------------------------------------

The package automatically handles both camelCase and snake_case field names.
